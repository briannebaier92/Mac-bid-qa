# Mac-bid-qa
Comprehensive QA strategy and manual testing suite for the MAC.BID mobile application, focusing on user experience, functional validation, and bug lifecycle management.
MAC.BID Manual QA Testing Suite

Project Overview

This project serves as a comprehensive manual testing repository for the MAC.BID auction platform. The goal is to ensure a seamless user experience by validating core functional flows, UI consistency, and edge-case scenarios across both web and mobile interfaces.

🎯 Testing Objectives

Functional Validation: Ensuring the bidding engine, user registration, and search filters work according to business logic.

User Experience (UX): Identifying friction points in the auction navigation and checkout processes.

Regression Testing: Verifying that existing features remain stable after platform updates.

Cross-Browser/Device Compatibility: Testing responsiveness on Chrome, Safari, and mobile app views.
